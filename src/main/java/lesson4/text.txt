Модификаторы доступа:

public - Видна абсолютно везде

private - Переменная полностью закрыта извне класса,
только разработчик класса может изнутри класса может менять, никто извне не имеет доступ к переменной, ...
private int age;
При инициализации (в конструкторе написать):
this.age = 10;
Внутри класса можно с этой переменной работать.
Снаружи никто не может обратиться к этой переменной.

protected - это по сути package default, но кроме того, что вы можете достучаться в рамках пакета + еще
классы-наследники. Если класс находится в другом пакете и он еще и наследован, то protected тоже будет работать.
Очень удобно при работе с библиотеками.

(package default) - идет по умолчанию, вручную не устанавливается.
Если опустить модификатор доступа и его не писать, то переменная будет видна всем классам, но только в рамках
того пакета, где лежит текущий класс.
